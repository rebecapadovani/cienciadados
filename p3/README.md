# Referência bibliográfica do artigo lido
> Giovanni Scardoni, Michele Petterlini, Carlo Laudanna, Analyzing biologic network parameters with CentiScaPe, Bioinformatics , Volume 25, Issue 21, 1 November 2009, Pages 2857–2859, https://doi.org/10.1093/bioinformatics/btp517.

# Resumo
> O artigo tem como objetivo mostrar como a ferramenta CentiScape é versátil e fácil de usar em bioinformática quando a análise de rede é baseada em centralidade. Os autores demonstram a partir de experimentos realizados com dados de kino-phosphatome humano, que são um compilado de bancos de dados públicos (HPRD, BIND, DIP, IntAct, MINT e BioGRID). O artigo contém os resultados dos cáluclos de várias centralidades de rede, como Distância Média, Diâmetro, Grau, Tensão, Intermediação, Radialidade, Proximidade, Valor Centroide e Excentricidade dos dados de experimentos e algumas análises biológicas provenientes dos resultados gerados.
